== Description ==

Single block containing multiple elements, It will enhance your website visibility and your needs. With this, you will change page layout (Fixed, Semi), set All Background Property (like  color, overlay, opacity, gradient style), set different Border, set height, width and set All Padding Property.

Multipurpose Gutenberg <a href="http://bit.ly/2TzFTNP" target="_blank"> Block Demo </a>

== Why we built this plugin?== 

With Default Gutenberg block have limited elements, you can not create some unique and specific business need layouts. But "Multipurpose Gutenberg block" give you more control to quickly create and launch any kind of site layouts you want!

== Key Features ==

**Page Layout**
    
With this option, you can set two different page layout fixed and semi.

- Fixed -  Set page layout with fixed width 1200px
- Semi -  Set page layout with semi width 85% 

**Background Setting**

There is a list of background Property available in Multipurpose Gutenberg block. with this, you can upload a custom image, set the background color and choose gradient background also. 

The following settings are available background:

= Background Setting =
- Background Size ON - Set background-size "Cover"
- Background Attachment ON - Set background-attachment "Fixed"
- Select Position 

= Overlay =  
- Overlay Color
- Background Opacity

= Background Color =

= Gradient Background =
- Select Gradient Type
	
**Border Setting**

With this option, you can set visible block border for page content and It will more readable page with help of border setting. The following settings are available for the border.

- All Border
- Top Border
- Right Border
- Bottom Border
- Left Border
- Border Color
- Border width
- Border-radius

**General Setting**

With this option, you can set custom width, height, and padding of the specific block. The following settings are available background: 
 
- Set Width(%)
- Set Height(px)
- Padding Setting (px)
- Padding Top
- Padding Right
- Padding Bottom
- Padding Left

**Advanced**

The use of Additional CSS classes can help to modify the look of the blocks used in Gutenberg.

This section describes how to install the plugin and get it working.

e.g.

1. Upload `multipurpose-block` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
